* 기간 : 2023.04 ~ 2023.06
* 사용도구 : Unreal Engine5 , Github , C++

## 게임 Play 방법
### PC 플레이어
- W,A,S,D : 이동
- Q : 구르기
- L-Shift : 슬라이딩
- Space Bar : Fly
- E : 상호작용
- R : 쓰레기 봉투 발사
- G : 아이템 버리기
- Mouse Left-Click : Fire
